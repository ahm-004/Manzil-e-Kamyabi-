Firebase Setup Guide 🔧

Ek baar setup karein — sab kuch live ho jayega

Step-by-Step Setup (5-10 minutes)
1

Firebase Project Banayein

Browser mein kholo: console.firebase.google.com
→ Add project click karein
→ Naam dein: manzil-e-kamyabi
→ Google Analytics: OFF kar dein
→ Create project

2

Authentication Enable Karein

Firebase Console mein:
→ Left menu: Authentication
→ Get started
→ Email/Password → Enable → Save
→ Users tab → Add user
→ Apni email aur password enter karein (yeh admin login hoga)

3

Realtime Database Banayein

Firebase Console mein:
→ Left menu: Realtime Database
→ Create database
→ Location: us-central1 (ya koi bhi)
→ Start in test mode → Enable

4

Firebase Config Copy Karein

Firebase Console:
→ Gear icon (⚙️) → Project settings
→ Scroll down: Your apps
→ Web app icon (</>) click karein
→ App nickname: mek-web → Register
→ firebaseConfig object copy karein

// Aapka config kuch aise dikhega: const firebaseConfig = { apiKey: "AIzaSyXXXXXXXXXXXXXXXXXXX", authDomain: "manzil-e-kamyabi.firebaseapp.com", databaseURL: "https://manzil-e-kamyabi-default-rtdb.firebaseio.com", projectId: "manzil-e-kamyabi", storageBucket: "manzil-e-kamyabi.appspot.com", messagingSenderId: "123456789012", appId: "1:123456789012:web:abcdefghijklmnop" };
5

Config Paste Karein Dono Files Mein

Is admin.html file ko Notepad/VS Code mein open karein.
PASTE_YOUR_APIKEY wali line dhundein (top of JS section).
Apna actual config wahan paste karein.

Phir index.html (main website) mein bhi yahi karein.

6

Database Security Rules Set Karein

Realtime Database → Rules tab → yeh paste karein → Publish:

{ "rules": { "tests": { ".read": true, ".write": "auth != null" }, "results": { ".read": "auth != null", "$testId": { "$sid": { ".write": "!data.exists()" } } }, "papers": { ".read": true, ".write": "auth != null" }, "announcements": { ".read": true, ".write": "auth != null" }, "mcqBank": { ".read": true, ".write": "auth != null" }, "mcqPending": { ".read": "auth != null", ".write": true }, "liveTests": { ".read": true, ".write": "auth != null" } } }
7

Netlify par Deploy Karein (Free Hosting)

1. netlify.com → Free account banayein
2. Sites → Deploy manually
3. Apna folder (jismein admin.html aur index.html hain) drag & drop karein
4. Aapko milega: https://your-name.netlify.app
5. Student test link banega: https://your-name.netlify.app/test.html?id=TEST_ID

✅ Total Cost: ZERO — Firebase free tier + Netlify free tier = bilkul free!
